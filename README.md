uright-server
=============
A lightweight server for uRight iOS app.


Requirements
============
* redis (http://redis.io/)
* rq (http://python-rq.org/)
* Flask (http://flask.pocoo.org/docs/)

<pre>
apt-get install redis-server
pip install rq
pip install Flask
</pre>


Starting the server
===================
<pre>
python start-uright-server.py
</pre>
